### Here's the list of `Kali` Linux commands 
```
1. pwd             # Print working directory
2. ls -la          # List files with details
3. cd <dir>        # Change directory
4. sudo apt update && apt upgrade -y  # Update Kali
5. sudo apt install <package>         # Install tools (e.g., nmap, scapy)
6. ifconfig / ip a  # Show network interfaces
7. netstat -tuln   # Show listening ports
8. ping <host>     # Send ICMP echo requests
9. man <command>   # Show manual page
10. history        # Show recent commands
```
